# Agency Landing Page Task Manager

## Overview
A simple task manager application for an agency landing page using HTML, CSS, and JavaScript.

## Features
- Add tasks with text
- Toggle tasks as complete/incomplete
- Delete tasks
- Persist tasks in localStorage
- Clean and simple UI

## Running the Application
1. Download the files to your local machine.
2. Open `index.html` in a web browser.
3. You can add, toggle, and delete tasks as needed.

## Technologies Used
- HTML
- CSS
- JavaScript
